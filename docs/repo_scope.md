# Repository Scope

This repository documents the reverse-engineering state of *Lands of Lore I* and publishes the tooling and structured outputs needed to reproduce the findings.

Included:

- format notes
- extraction tools
- inventory and analysis JSON
- edition comparison notes

Excluded from the initial repo:

- raw retail game files
- full extracted game-media dumps
- superseded debug scripts

The goal is to make the work usable and reviewable without turning the repo into a copyrighted asset mirror.
